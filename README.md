# Projet-M431
Conventions de nommage : snake_case

## But du projet
- Jeu de parcours

## Raccourcis clavier
Déplacement : AD / ⬅️➡️  
Regénération : Q / -  
Attaque : Clic gauche  
Attaque pogo : S / ⬇️  
Intéragir : E / .  
Slide : touche MAJ  
Saut : W / ⬆️ / espace  
Double saut : 2x espace  

## Objets présents dans le jeu 
Potion : sert à se regénérer  
Piques : inflige des dégâts  
Coffre : peut être ouvert pour récupérer l'objet à l'intérieur  
Épée : débloque l'attaque et l'attaque pogo  
Parchemins : débloque des habiletés (dash, saut mural, double saut)
